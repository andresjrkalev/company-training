# Company training

Company training courses with participants.

## Run application

In terminal execute

```shell
./gradlew bootRun
```

## Run tests

```shell
./gradlew test
```

Report is located: **build/test-coverage-report/index.html**

## Database

Located: http://localhost:8080/h2-console

## Documentation

Located: http://localhost:8080/swagger-ui/index.html
